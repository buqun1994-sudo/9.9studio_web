# 9Studio

Static landing page for `9studio.fun`, prepared for GitHub Pages deployment.

## Files

- `index.html`: main landing page
- `styles.css`: visual styles and responsive layout
- `404.html`: simple fallback page
- `CNAME`: GitHub Pages custom domain binding

## Deploy

1. Push this repository to GitHub.
2. Open repository `Settings -> Pages`.
3. Set source to `Deploy from a branch`.
4. Choose `main` and `/root`.
5. Confirm custom domain is `9studio.fun`.
